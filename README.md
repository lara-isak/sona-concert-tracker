# SONA

A cyberpunk-themed concert tracking dashboard built with Next.js and shadcn/ui.

## Features

- Track concerts and shows with detailed information
- Filter by city, vendor, attendance status, and year
- Search functionality across shows, cities, and venues
- Beautiful cyberpunk-inspired UI with neon accents
- Responsive design for mobile and desktop

## Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI components
- **Lucide React** - Icons
- **Vercel Analytics** - Analytics

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
sona/
├── app/              # Next.js app directory
│   ├── page.tsx     # Main concert tracker page
│   └── layout.tsx   # Root layout
├── components/       # React components
│   └── ui/          # shadcn/ui components
└── lib/             # Utility functions
```

## Build

To create a production build:

```bash
npm run build
```
